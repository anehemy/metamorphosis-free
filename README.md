# Metamorphosis Free

The free Claude plugin marketplace for [Metamorphosis Worldwide](https://metamorphosis-usa.com) —
the giveaway tier of the Customer Clarity AI System (CCAIS).

## Install (works in the Claude app, claude.ai, and Claude Code)

Open the plugins screen — in the Claude app / claude.ai it's the **+** menu (or
Settings) → **Plugins**; in Claude Code it's the `/plugin` command — then:

1. **Add marketplace:** `anehemy/metamorphosis-free`
2. **Install** the **Customer Clarity AI System (Free)** plugin.

No GitHub account needed. Then ask *"who is my customer really?"* or paste a
call transcript, and the skill runs.

Claude Code equivalent:

```
/plugin marketplace add anehemy/metamorphosis-free
/plugin install mw-ccais-free@metamorphosis-free
```

## Prefer a single file? (claude.ai alternative)

You can also skip the marketplace and upload the skill as a file:
download **`mw-ccais-free-avatar-scan.zip`** from this repo, then in Claude go
to **Customize → Skills → + → Upload a skill**.

## What's inside

| Plugin | Skill | What it does |
|--------|-------|--------------|
| `mw-ccais-free` | `mw-ccais-free-avatar-scan` | Builds one customer avatar from your real call transcripts, in your customers' own words. Pulls from Fathom or Microsoft 365 when connected, or works from pasted transcripts. |

More free tools will be added over time. The full paid Customer Clarity AI
System lives in a separate marketplace.
